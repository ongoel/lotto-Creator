### **[오늘의 로또]** 워크플로우

### **[개발 목표]** 로또번호를 랜덤으로 생성해주는 웹 앱

* **[앱의 이름]**
    * 오늘의 로또

* **[구현을 위한 도구]** (google AI pro 요금제로 사용가능한 도구와 무료로 사용가능한 도구만 사용)
    * 안티그래비티
    * 깃 허브
    * cloud flare
    * stitch(구글 AI PRO 요금제로 사용가능 1일 사용량 제한으로 신중히 사용하여야 함)

* **[앱 구현방식]**
    * 안티그래비티를 사용하여 앱의 구조(코드)를 생성
    * 깃 허브를 사용하여 앱의 구조(코드)를 관리
    * 깃 허브와 cloud flare를 사용하여 앱을 배포

* **[앱의 기능]**
    * 로또를 1게임 또는 5게임을 선택하면 랜덤으로 로또번호를 생성
    * 생성된 로또번호를 화면에 표시
    * 생성된 로또번호를 스크린샷으로 저장하는 기능(버튼) 구현
    * 로또번호를 생성할 때 귀여운 캐릭터가 로또번호를 생성하는 애니메이션 구현
    * 로또 번호 생성 이력을 저장하고, 이력에서 로또번호를 선택하면 해당 로또번호를 화면에 표시

* **[앱의 구조]**
    * 메인 화면
        * 로또번호 생성 버튼
        * 로또번호 표시 영역
        * 로또번호 저장 버튼
        * 로또번호 공유 버튼
    * 설정 화면
        * 로또번호 개수 선택
        * 로또번호 생성 애니메이션 설정
        * 로또번호 저장 위치 설정
        * 로또번호 공유 설정

* **[앱의 디자인]**
    * 모바일 화면에 최적화된 디자인
    * 다크모드와 라이트모드 전환가능한 디자인

* **[앱의 기능 추가 계획]** (추후 구현 예정)
    * 생성된 로또번호를 실제 삿는지 않삿는지 체크하는 기능 구현(구입 회차정보 입력 가능하게 구현)
    * 체크된 로또번호 정보를 이용하여 해당 회차 로또당첨번호와 대조하여 당첨확인하는 기능 구현
    * 사용자가 생성한 로또번호중 실제 당첨된 로또가 있는지 관리자가 확인할 수 있는 기능 구현
